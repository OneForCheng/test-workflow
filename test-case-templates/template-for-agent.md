## 字段说明

| 字段名 | 说明 | 默认值 / 约束 |
|--------|------|---------------|
| FuncNo | 功能点编号，同一功能点下所有用例共享同一编号 | 格式为 `F{n}`，从 F1 开始递增。同一功能点的第二条用例起留空 |
| FuncDesc | 功能点名称，简要描述该功能点 | 同一功能点的第二条用例起留空 |
| CaseNo | 用例编号，全局唯一 | 格式为 `TC{nnn}`，从 TC001 开始递增，三位数字补零 |
| CaseDesc | 用例描述，概括该条用例的测试目的 | 必填，简明扼要 |
| Steps | 测试步骤，编号列表描述具体操作 | 必填，每步一行，格式为 `1. xxx` |
| Expected Result | 预期结果，编号列表与步骤对应 | 必填，每条一行，格式为 `1. xxx` |
| Status | 用例状态 | 默认为 `Draft`，可选值：Draft / Approved |
| Priority | 优先级 | 可选值：Critical / High / Normal，默认 Normal |
